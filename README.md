## 简易登录注册系统  
**注册成功** -- “恭喜，注册成功”  
**注册重复** -- “该用户名已被注册”  
**登录成功** -- “登录成功”  
**登录名不存在** -- “该用户仍为被注册”  
**密码不符合** -- “用户名或密码错误”  
**发生错误** -- “注册失败”/登录失败
